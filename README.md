# Git_GH_Flow_Practice

## Process
```shell
# 1. Local
$ git branch feature/{featurename}
$ git switch feature/{featurename}

# do work

$ git status
$ git add {filename}
$ git commit
$ git push -u origin feature/{featurename}

# 2. Open pull request on github.com
# 3. Confirm and Merge
# 4. Delete branch on both local and remote
```
