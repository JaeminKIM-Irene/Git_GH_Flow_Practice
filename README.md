# Git_GH_Flow_Practice

## Process
```shell
# 1. Local
$ git branch feature/{featurename}
$ git switch feature/{featurename}

# 2. do work
$ git status
$ git add {filename}
$ git commit
$ git push -u origin feature/{featurename}

# 3. Open pull request on github.com

# 4. Confirm and Merge

# 5. Delete branch on both local and remote

# 6. Update local main branch
$ git pull origin main
```
